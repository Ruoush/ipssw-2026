Before the Workshop

3 things to do before you arrive

Set up the tools you'll need for the workshop. Takes about 10 minutes — and you only have to do this once.

✅  No coding experience needed.  |  ✅  All tools are free.  |  ✅  Works on any laptop.
💻 A laptop is strongly preferred. The hands-on activities are designed for a full keyboard and browser. If you only have a tablet or phone, reach out to your facilitator — you'll be paired with a partner for the build activity.
1
Create an AI Account
You'll use this to generate your simulation tool during the workshop

The workshop uses agentic AI tools to generate HTML simulation tools from plain-language descriptions. You'll need a free account with at least one of the options below before you arrive.

Claude Recommended
Go to claude.ai → Sign Up → free tier is available. Claude tends to generate complete, clean HTML in one shot, which makes the workshop workflow smoother. This is the AI used in the live demo.
ChatGPT
Go to chat.openai.com → Sign Up → free tier available. Widely used and works well with the prompt templates from this workshop.
Gemini
Go to gemini.google.com → Sign in with your Google account → free to use. A good option if you're already in the Google ecosystem.
Note: All three AI tools work with the prompt templates in the workshop. You only need one account.
2
Install a Code Editor
Where you'll paste AI output, save it as an HTML file, and open it in a browser

You'll copy code from the AI tool and paste it into a file, then save it as .html. A code editor makes this easier and reduces the chance of accidentally breaking the file. You'll need to install one before the workshop.

Once saved, open your HTML file by double-clicking it — it will open in your default browser. You can also drag the file into any browser window.

⚡
Cursor Recommended
An AI-native code editor. Download free at cursor.com. Has a free tier and makes it easy to paste, preview, and iterate on HTML files. This is what most participants use during the workshop.

Setup steps:

  1. Open Cursor → File → Open Folder → create or choose a folder on your Desktop (e.g., "my-sim-tools")
  2. Create a new file: click the New File icon in the sidebar → name it anything ending in .html
  3. Paste your AI-generated code into the editor area
  4. Save: Cmd+S (Mac) or Ctrl+S (Windows)
  5. Double-click the file in Finder/Explorer to open it in your browser — or drag it into a browser window

📄
TextEdit (Mac) / Notepad (Windows)
Works as a last resort — no download needed. But you must take an extra step:

Mac TextEdit: Go to Format → Make Plain Text before saving. Save the file with a .html extension.

Windows Notepad: In the Save dialog, change "Save as type" to All Files, then name the file with quotes: "my-tool.html" — this prevents Windows from adding .txt automatically.
3
Terminal Access — optional
Not required for installation or opening files — here if you want it

A terminal lets you type commands to control your computer. You don't need it for this workshop — everything can be done by double-clicking files and using the menus in your code editor. It's included here as a reference for those who want it.

Show terminal instructions — Mac & Windows

A terminal is a text-based interface to your computer — you type commands instead of clicking icons. The commands below navigate to your Desktop and open an HTML file. Replace your-filename.html with the actual name of your file.

🍎 Mac — Terminal

How to open: Press Cmd + Space, type "Terminal", press Enter.

When it opens, you'll see a blinking cursor — it's ready for a command:

Terminal — zsh
yourname@MacBook ~ % █

Navigate to your Desktop, then open your file:

Terminal — zsh
yourname@MacBook ~ % cd ~/Desktop
yourname@MacBook Desktop % open your-filename.html
# Your default browser opens the file automatically
🪟 Windows — Command Prompt

How to open: Press Win + R, type cmd, press Enter.

Command Prompt
C:\Users\YourName> cd Desktop
C:\Users\YourName\Desktop> start your-filename.html
:: Your default browser opens the file automatically
